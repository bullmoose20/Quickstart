Logscan Columns
Finished: timestamp of the run finishing.
Runtime: total run time.
Config: config name detected for the run.
Command: sanitized Kometa command line.
Counts (W/E/T + M/S/EP/TOT): Warnings, Errors, Tracebacks plus library totals (Movies, Shows, Episodes, Total) for the run.

Section Runtimes: run time totals per section.
